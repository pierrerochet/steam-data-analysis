# Description
Quelques analyses sur les données de la plateforme de jeux vidéos **Steam**.

# Accéder aux analyses
Le projet se présente sous la forme d'une page web que vous trouverez à l'adresse suivante :  
https://pierrerochet.github.io/steam-data-analysis/

# Données
Les données ont été extraites du dataset **steam** téléchargeable à cette adresse :  
https://www.kaggle.com/nikdavis/steam-store-games#steam_description_data.csv

# Graphiques
Les graphiques ont été réalisés en python grâce à la librairie [**Ploty**](https://plot.ly/python/)
